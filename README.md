# Ski-Jumping
This replicates the analysis in 
This repository contains replication files for the empirical results in Haugen, Kjetil K. & Musau, Andrew (2025). Do Subjective and Compensatory Scores Matter? An Empirical Analysis of Ski Jumping Competitions. Sports Economics Review (SER), 100061. doi: 10.1016/j.serev.2025.100061.

File Descriptions

File descriptives.do
Reproduces the results presented in Table 1, Table 2, and Table 3 of the paper.

File R2_ALL_REV.do
Replicates the results reported in Table 4 and recreates Figure 1 and Figure 2.

File Rank10_all.do
Reproduces the results shown in Table 5, Figure 3, Figure 4, and Table 6.

File Ranks_all.do
Replicates the results presented in Table A1, Figure A1, and Figure A2 in the Appendix.
